    数据获取

    本项目的数据均来源于“汽车之家”网址的口碑论坛。利用Python爬虫技术，通过selenium模块驱动浏览器，模拟人工操作，并借助“BeautifulSoup”工具包进行数据解析。

    自然语言处理

    本项目利用长短期记忆神经网络 (LSTM) 进行情感分析，情感极性包括正面、负面两类。评论的多标签分类则利用pytorch搭建了TextRCNN与TextCNN模型, 并结合标签共现性与注意力机制进行改进，分为空间, 动力, 操控, 能耗, 舒适性, 外观, 内饰, 性价比， 配置, 续航, 安全性等十八类, 最总的输出为 利用“投票机制”结合多个模型的综合结果。关键词提取使用了“jieba”工具包中的关键词提取工具，根据句子的长度不同分别利用TFIDF或者TextRank算法进行关键词提取。

    图表可视化

    情感极性与类别的数据展示借助了基于 JavaScript 的开源可视化图表库ECharts实现。词云则使用了“wordcloud”工具包实现。
